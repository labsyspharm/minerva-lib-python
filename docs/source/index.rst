Installation
-------------

This documentation is for the python library :mod:`minerva_lib`.
First, download the source code here:

.. code-block:: bash
    
    git clone https://github.com/sorgerlab/minerva-lib-python
    cd minerva-lib-python

Then, install the library:

.. code-block:: bash

    pip install .
